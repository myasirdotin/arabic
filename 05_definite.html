📌 Definite vs. Indefinite Nouns (المعرفة والنكرة)

التعريف والتنكير — The Arabic Definiteness System

Arabic nouns are either definite (معرفة) (specific) or indefinite (نكرة) (general). This affects their endings and sentence structure.

✅ Key Differences:

Feature Definite (المعرفة) Indefinite (النكرة)
Meaning Specific (the) General (a/an)
Marker الـ prefix Tanween (ـٌ، ـً، ـٍ)
Example الكتابُ (the book) كتابٌ (a book)
I'rab ـُ / ـَ / ـِ (no nunation) ـٌ / ـً / ـٍ (with nunation)

🎯 Types of Definite Nouns:

1. With "Al-" (أل التعريف)

الـ + كتاب → الكتابُ

قرأت الكتابَ
I read the book (definite object)

2. Proper Names (أعلام)

محمدٌ ← محمدُ (when definite)

رأيتُ محمداً
I saw Muhammad (treated as definite)

📘 Solar & Lunar Letters (التعريف الشمسية والقمرية)

Sun Letters (حروف شمسية)

الـ assimilates with the first letter (14 letters):

ت ث د ذ ر ز س ش ص ض ط ظ ل ن

ال + شمس = الشَّمس (not "الشمس")

Moon Letters (حروف قمرية)

الـ remains clear (all other letters):

ال + قمر = القَمَر (pronounced fully)

🧠 Tanween Rules:

Case Indefinite Ending Definite Equivalent
Nominative ـٌ (un) ـُ
Accusative ـً (an)* ـَ
Genitive ـٍ (in) ـِ

*ـً becomes ـاً on "alif-less" nouns (e.g., اسم → اسمًا)

📝 Definiteness in Sentences:

Indefinite: هذا كتابٌ جديدٌ
This is a new book (general)

Definite: هذا الكتابُ جديدٌ
This book is new (specific)
